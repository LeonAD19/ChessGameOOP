# How to compile & run:
To compile: ```bash
javac -d bin src/**/*.java

To Run with the GUI:
java -cp bin gui.ChessGUI

To Run with Console(Terminal):
java -cp bin game.Game

To Run without Compiling on IDE:
Look for and click run button

# Achieved Features:
- 8x8 chessboard rendered in GUI
- All chess pieces displayed correctly 
- Piece selection & movement (click-to-move)
- Piece capturing logic
- King capture triggers Game Over pop-up
- Reset game functionality
- Full move validation 
- Check & checkmate detection
